ChatGPT — 아이디어/대본

니치 리서치 → 훅/스크립트 템플릿 → 숏폼 스토리보드.

  • Hook 5종 템플릿 (문제제기/반전/숫자/비교/ASMR)
  • 30초/60초 스크립트 자동 생성 프롬프트

Suno — 음악/효과음

장르/무드/구성 프롬프트, 보컬/인스트루먼트 버전.

  • 인트로 5초 스팅/브레이크 만들기
  • 루프 가능한 BGM 추출

Midjourney — 이미지/썸네일

아이캐치, 브랜드 컬러, 문자 선명도, 16:9/9:16 가이드.

  • 썸네일 3안 빠른 탐색 프롬프트
  • CI 색상/폰트 고정 프리셋

Runway/Veo — 영상 생성

씬 스크립트 5초 단위, 카메라 워크/라이트 프롬프트.

  • 쇼츠용 9:16 안정 프리셋
  • 컷 간 시각 톤 일치시키기

CapCut — 편집/자막/사운드

L컷/제이컷, 오토캡션 정리, LUT/텍스트 프리셋.

  • 비트 싱크 단축키/리듬 패턴
  • 템플릿 프로젝트 제공(.cptemplate)

업로드 & 분석

메타데이터/해시태그/챕터, CTR·Retention 기본 해석.

  • 제목/설명/태그 자동 생성 폼
  • 초반 1시간 체크리스트

FAQ

완전 초보도 가능한가요?

가능. 툴 설치/가입부터 시작해서, 템플릿에 값만 채워도 결과물이 나오도록 구성했어.

저작권/라이선스는 어떻게 다루나요?

각 툴의 이용 약관과 유튜브 가이드라인을 함께 점검하고, 상업적 사용시 주의할 포인트를 체크리스트로 제공해.

Shorts와 16:9 중 뭘 먼저 만들까요?

리텐션 학습엔 Shorts가 빠르고, 브랜드 서사는 16:9가 유리. 수업에선 두 포맷을 병행해 템플릿을 나눠 제공해.
